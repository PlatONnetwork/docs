ATON Wallet User Manual

ATON Download


Click to download the latest version of ATON

Create Wallet

1. Click [Create Wallet].
2.Select type (Normal or HD). Set the wallet name, and set the wallet password.
3. Click the [Create] button to create a wallet.
4. After the wallet is successfully created, you can click [Start to Backup] to start the backup, or click [Skip] to perform backup later.

Note
1. Password is your credentials to operate the wallet. PlatON does not store your password, make sure you keep it in mind.
2. We strongly recommend you to back up your wallet after creating it.

Wallet backup


1. After creating the wallet, click [Start Backup] button for wallet backup. Or you can find it in [My-Wallet Management] module.
2. Wallet password is reeuqired for backup. After the password verification is passed, the system will display 12 English words as mnemonic words.
3. Copy the mnemonic and keep it in a safe place. Click [Next] to verify the mnemonic. If the verification succeeds, the wallet backup is completed.

Note
1. Never lose your mnemonics,keep it in a safe place.

Import wallet

Import Using KeyStore


1. Click [Import Wallet-Keystore].
2. Paster the file, or scan the QR code to get the keystore file.
3. Set the wallet name and enter the password for this wallet.
4. Click the [Start Import] button. After the password verification is passed, the wallet is successfully imported.

Note
1. Password is need when import ing wallet Keystore.
2. Your wallet password is not stored on the server. If you forget the password, please use mnemonic and private key to import.
3. You can also click the "..." button on the top corner to import wallet.

Import Using Mnemonics


1. Click [Import Wallet-Mnemonic].
2. Fill in the mnemonics correctly and in the correct order.
3. Select type (Normal or HD). Set the wallet name and wallet password.
4. Click the [Start Import] button. After the mnemonic verification is passed, the wallet is successfully imported.

Note
1. Password is your credentials to operate the wallet. PlatON does not store your password, make sure you keep it in mind.
2. Store your mnemonics in a safe place.
3. You can also click the "..." button on the top corner to import wallet.

Import Using Private Key


1. Click [Import Wallet-Private Key].
2. Paster the file, or scan the QR code to get the private key.
3. Set the wallet name and wallet password.
4. Click the [Start Import] button. After the verification is passed, the wallet is successfully imported.

Note
1. Password is your credentials to operate the wallet. PlatON does not store your password, make sure you keep it in mind.
2. Store your private key in a safe place.
3. You can also click the "..." button on the top corner to import wallet.

Import observed wallet

For details about importing observation wallets, please refer to the instructions of “Set observed wallet and cold wallet”

My Wallet


1. Click the [Wallet] module, you can see your wallets.
2. ① Show the total number of LAT for all wallets.
3. ② Click different wallets to switch wallets.
4. ③ Click to select any child wallet from this group of HD wallets.
5. ④ Shows your balance for particular wallet. If there is a locked balance, the locked amount will be displayed below.
6. ⑤ Display all transaction records of the selected wallet.
7. ⑥ You can transfer LAT to other accounts.
8. ⑦ Show the selected account address and address QR code, which can be transferred to LAT.
9. ⑧ Click [...] to create,import and select wallet.
10. ⑨ Show the wallet detail.

Select Wallet


1. Click [...] , then click [Select Wallet].
2. Can be classified and filtered by "All", "HD", "Normal", and can also be used for fuzzy search by wallet name or precise search by wallet address.
3. Find the wallet to be selected, click the wallet box.
4. My Wallet page automatically switches to the wallet you just selected.

Receive


1. Select wallet and click the [Receive] column to check the wallet address and QR code.
2. Click the wallet address to copy the address.
3. Click [Save Picture and Share] to save the picture to the local album.
4. Other parties can use this address and scan this QR code to make LAT transfers.

Note
1. This APP address and QR codes only accept assets for PlatONnetwork, do not deposit other network assets to this purse.
2. Please make sure you deposit assets for correct network.

Send


1.Select wallet and click the [Send] column.
2. Enter a valid wallet address, or scan the QR code to get address.
3. Enter amount.
4. Set transaction fee rate.
5. Click [Next] to confirm the transaction information, and click [Send] to proceed the transaction. Click [Cancel] to cancel this transaction.

Validators

Validators list


1. Click [Delegate-Validators], you can view all validators.
2. Defaultly, the nodes are sorted by rank. They can also be sorted by Delegated/Delegators/Expected delegation annualized yield.
3. Active node: validators participating in the consensus election in the current settlement cycle.
4. Candidate node: validators does not participate in consensus election in the current settlement cycle.

Validator Detail


1. Select a validator, you can check the reward related informaiton, like expected delegation annualized yield, reward ratio and total reward.
2. Also you can check node basic information-delegators, blocks related,etc.
3. If you want to delegate to this node, you can click the [Delegate] button to operate.

Note:
1. Foundation funded nodes do not accept delegations.
2. The locked node does not accept delegation

Delegate

Submit Delegation


1. Select a node to delegate.
2. Select the wallet.
3. You can select locked balance to delegate if there is.
4. Enter the amount, and it must not be less than 10LAT.
5. Enter the correct password.
6. You can delegate to the same nodes without limit.

Note
1. The wallet associated with nodes is not allowed for delegation.

My Delegations


1. Click [My Delegation] to check your uncalmied reward, total reward and claim records.
2.Select specific wallet to check the details.
3.Click the [Claim] button to get your rewards.

Withdraw Delegation


1. Click [My Delegations-Details-Withdraw].
2. Enter the number,it should be greater than or equal to 10LAT. Enter the correct password to proceed the process.
3. You can click the [Delegated] drop-down box to select different types of delegation.

Note
1. If the remaining delegations has been less than 10LAT, it will be automatically returned.
2. When withdrawing all delegations, Claim rewards automatically.

Set observed wallet and cold wallet

Preparation:

  1. Prepare two mobile phones and install the latest ATON wallet version.
  2. Make sure that one phone is never connected to the Internet. It is recommended to turn on the airplane mode and turn off WiFi to serve as a cold wallet device. The other one is connected to the network as an observation wallet device.

1. Create a wallet in offline environment, click [ Receive] to display the QR code. And this wallet is a cold wallet.
2. Click [Import Wallet-Observe Wallet] for the mobile phone connected to the network, scan the QR code of the mobile phone not connected to the Internet or enter the wallet address, and click [Finish]. Then the wallet is set successfully.

Note
1. Observation wallet: Connected to network, but there is not the private purse. If you need to conduct transactions, you need to cooperate with the cold wallet operation.
2. Cold wallet: Not connected to the Internet but has a private key, which needs to be kept in a safe place.

Transaction with cold wallet

If you have set up observed wallets and cold wallets, you can initiate transfers, delegations, withdrawals and other operations on the observation wallets, and use the cold wallets cooperate to sign. The following is an example.

1. Observed wallet operation


(1) Select an observate wallet and swich to the send Page, enter the information, proceed the transaction.
(2) Click the [Next] button, and the page displays the QR code information to be scanned by the cold wallet.

2. Cold wallet operation

(1) Click Scan in the upper right corner, or click the [Offline Signature] button of the specified wallet to scan and observe the QR code information displayed on the wallet page.
(2) The page displays the detailed information of the transaction. If the confirmation is correct, you can click [Next] and enter the wallet password.
(3) After the wallet password is entered correctly, the signed transaction QR code information will be displayed on the page for the observed wallet to scan.

3. Observed wallet operation


(1) Click [Next] buttonin step 1.
(2) Click the scan icon and scan the QR code on the cold wallet page in step 2 to get the signature data.
(3) Click [Send Transaction] to send the transaction to the blockchain network.

Note
1. Jointly use observed wallet and cold wallet to do transactions.
2. Please do not deposit assets to the observing wallet address that you have cannot control.

Settings / Help and Feedback


1. You can make related settings
-Node Settings: Select the network to be connected.
-Reminder Threshold:Remind when the amount of transfer exceeds the set amount.
-Resend Reminder:Remind when the transaction are the same as the one in last 2 housrs.
-Fingerprint / Face Unlocking: After opening, fingerprint or face unlock is required every time you enter the app.
-Language: Chinese and English can be selected to switch.
2. Select [My-Help and Feedback] You can view APP usage help and frequently asked questions. If you don't get the answer you want, you can submit feedback and we will respond as soon as possible.
